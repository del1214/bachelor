# 安装

> 环境要求：
> nodejs\git(svn)
node.js下载地址：https://cdn.npm.taobao.org/dist/node/v14.15.1/node-v14.15.1-x64.msi

1.  从`git`/`svn`上克隆脚手架
``` shell
git clone http://43.250.238.158/framework/vue-admin-scaffold.git
```

2. 安装yarn
``` shell
npm i yarn -g
```

3. 安装依赖
``` shell
yarn 或者 npm install
```

4.  运行开发环境
``` shell
yarn serve
``` 

5.  打包代码（输出生产环境包）
``` shell
yarn build
```
