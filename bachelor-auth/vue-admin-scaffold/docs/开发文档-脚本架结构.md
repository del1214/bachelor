# 脚本架结构

- `package.json` 项目元数据
- `vue.config.js` Vue 生产配置和开发环境配置
- `.eslintrc.js` ESLint配置文件
- `.babel.config.js` Babel配置文件
- `public` 静态资源，不经过webpack处理
- `src` 工程源码
  - `assets` 静态资源
    - `fonts` 字体
    - `images` 图像
    - `styles` 全局样式
  - `components` 公共组件
  - `directive` 自定义指令
  - `filter` 自定义过滤器
  - `lib` 通用库
  - `config` 项目配置文件
  - `util` 工具库
  - `layout` 框架默认布局
  - `store` 数据仓库
  - `router` 路由
  - `mixins` 混入
  - `service` http服务
    - `index` 服务配置入口（请求拦截/响应拦截）
    - `api` 对axios封闭
    - `mock` mock数据
  - `views` 视图组件
  - `App.vue` 入口组件
  - `main.js` 入口文件
- `tests` 单元测试
